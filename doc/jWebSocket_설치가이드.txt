
* jWebSocket 설치 가이드 입니다.


1. /html5/WebContent/WEB-INF/doc/jWebSocket-0.9.5.zip 압축푼다. (위치 : C:\ )

2. c:/jWebSocket-0.9.5/conf/jWebSocket.xml 파일이 있음을 확인 한다.

3. set JWEBSOCKET_HOME=c:\jWebSocket-0.9.5  설정

4. 윈도우즈 재부팅  ( set JWEBSOCKET_HOME 적용을 위함.)
